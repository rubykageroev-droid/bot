+# Telegram-бот оценки внешности и знакомств (MVP)
+
+MVP-реализация бота по ТЗ:
+- регистрация и анкета
+- просмотр анкет
+- лайк/нейтрал/дизлайк
+- взаимный матч
+- лимит 20 оценок в день
+
+## Запуск
+
+1. Установить зависимости:
+
+```bash
+pip install -r requirements.txt
+```
+
+2. Создать `.env`:
+
+```env
+TELEGRAM_BOT_TOKEN=your_token_here
+DB_PATH=bot.db
+```
+
+3. Запустить:
+
+```bash
+python main.py
+```
+
+## Структура
+
+- `app/bot.py` — хендлеры Telegram и сценарии регистрации/меню
+- `app/db.py` — SQLite-слой и логика рейтинга/матчей/выдачи анкет
+- `app/keyboards.py` — клавиатуры
+- `app/config.py` — конфиг из env
+- `main.py` — entrypoint
+
+## Что входит в MVP
+
+- `/start` + выбор цели
+- пошаговое создание анкеты
+- выдача анкеты с фильтрацией (без себя, без уже просмотренных)
+- реакция `like/neutral/dislike`
+- матч при взаимном like
+- «Кто меня лайкнул» (только количество)
+
+## Ограничения и упрощения
+
+- Пол пользователя не запрашивается (в ТЗ не задан), поэтому фильтрация по `gender_pref` хранится и готова, но в MVP минимально задействована.
+- Рост за день в профиле пока не реализован (оставлен на следующий этап аналитики).
